## Starting the Project

1. If you don't already have it, install [node.js](https://nodejs.org)
2. Install dependencies:

```bash
$ npm install
$ npm run build
```

## Mock REST API

1. If you don't already have it, install [node.js](https://nodejs.org)
1. Start the server:

```bash
$ npm run server
```
## Running Development Environment

Don't forget to start the mock API server, so:

```bash
$ npm run server
$ npm run dev
```
